This is a micro-frontend project. Need run all micro-frontends and host project to see all experience

## Commands

install dependencies

```
pnpm i
```

run only this project

```
pnpm dev
```

run this project with host

```
pnpm build
```

```
npx serve dist -p 5001
```

run tests

```
pnpm test
```

```
pnpm test:e2e
```
